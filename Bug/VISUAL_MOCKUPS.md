## 🎨 Component Design Updates

### Typography Changes (Updated!)

**Common Name (Bug Name)**
- Style: `.capitalized` - Only first letter of each word capitalized
- Font: 36pt, black weight, default design
- Example: "House Fly" (not "HOUSE FLY" or "house fly")
- Line spacing: -4pt (tight)
- Color: White (#FFFFFF)

**Scientific Name (Latin Name)**  
- Style: `.uppercase` - All caps
- Font: 11pt, **regular weight** (not italic)
- Example: "MUSCA DOMESTICA"
- Tracking: 1.2pt
- Color: Dim gray (#888888)

## ✅ Accessibility

### Dynamic Type Support
- All text scales with system font size
- `.dynamicTypeSize(...DynamicTypeSize.xxxLarge)` limit on bug name
- `.fixedSize(horizontal: false, vertical: true)` for multi-line wrapping

### Color Contrast
- All text: White (#FFFFFF) on dark (#111111) = 18.17:1 ratio ✅
- Labels: #666666 on #111111 = 4.68:1 ratio ✅ (WCAG AA)
- Accent colors tested for visibility

### VoiceOver
- All cards labeled descriptively
- Lock icons announce "Locked content"
- Purchase buttons have clear labels
