## Key Interactions

### Purchase Flow
1. User taps one of the two option cards â†’ `selectedProductType` updates
2. User taps "Unlock Now" â†’ `purchase()` called
3. StoreKit sheet appears â†’ User completes purchase
4. Transaction verified â†’ `isPro` becomes true
5. View automatically updates (SwiftUI @Published binding)
6. Locked cards disappear, premium content appears

### Restore Flow
1. User taps "Restore Purchases"
2. `AppStore.sync()` called
3. All current entitlements checked
4. `isPro` updated if any valid purchase found
5. View updates automatically
